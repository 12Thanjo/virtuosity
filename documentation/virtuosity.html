virtuosity

Virtuosity is a free, open source game engine that is puts all of the systems that I have built / found together in one easy place. It is designed to make game development faster and easer by automatically doing all of the annoying stuff. Virtuosity is designed for the intended use in Electron.

Rendering

Virtuosity uses Babylonjs 4.1 for 3d rendering and Phaser-CE 2.15.1 for 2d rendering. Phaser 3.x is more feature rich and suposedly performs better, however it is not equiped for some features that are neccesary for Virtuosity. Phaser-CE is still getting support as well.

Objects

engine2d

engine2d manages all of the graphics rendered in 2D. It uses Phaser-CE 2.15.1 for rendering. engine2d uses Phaser as a base and builds off of it to automatically interact with the rest of the engine, as well as adding custom functions for added features and ease of use for the already existing ones. Phaser can be accessed directly, but it is only recommended if there is a specific feature that Phaser has that is not currently implemented into engine2d.
