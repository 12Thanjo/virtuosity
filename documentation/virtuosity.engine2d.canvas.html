ParameterTypeDescription
pollint(OPTIONAL) polling rate of the update function for engine2d (times per second). This must be set with the first canvas created. [default: 64]
loadfunction(OPTIONAL) function to complete while engine2d is automatically loading assets
createfunction(OPTIONAL) function to complete while engine2d has finisished automatically loading assets
xInt(OPTIONAL) X position of the canvas [default: 0]
yInt(OPTIONAL) Y position of the canvas [default: 0]
widthInt(OPTIONAL) Width of the canvas [default: screen width]
heightInt(OPTIONAL) Height of the canvas [default: screen height]
